# Scripts

Esta pasta contém os scripts utilizados no projeto de estimativa da turbidez da água
a partir de dados do Sentinel-2.

## Conteúdo

- eda_turbidez.py  
  Análise exploratória dos dados e estatísticas descritivas

- correlacao.py  
  Análise de correlação entre turbidez, parâmetros ambientais e bandas espectrais

- modelos_ml.py  
  Treinamento e avaliação dos modelos de Machine Learning

- mapas_turbidez.py  
  Geração dos mapas espaciais de turbidez estimada
